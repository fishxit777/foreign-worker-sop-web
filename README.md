# 外籍員工溝通 SOP 包網站

這是「外籍員工溝通 SOP 包」的獨立 landing page。

建議 GitHub Pages 網址：

```text
https://fishxit777.github.io/foreign-worker-sop-web/
```

## 檔案

- `index.html`：一頁式網站
- `assets/factory-line-translation.jpg`：工廠主視覺
- `assets/wanyutong-line-qr.png`：萬語通 LINE QR
- `.nojekyll`：讓 GitHub Pages 直接使用靜態檔

## 上架方式

1. 到 Settings > Pages
2. Source 選 `Deploy from a branch`
3. Branch 選 `main` / root
4. 等待 GitHub Pages 產生網址

## 定位

這個網站不是取代萬語通主站，而是低門檻產品入口。

SOP 包負責讓老闆先理解「每天重複溝通可以模板化」。

萬語通負責後續導入 LINE 群組，產生月費 SaaS 收入。
